# LLM-Safety-Reports

# 🛡️ AI Audit Lab | Mornielov
> ************************** <

Bienvenido al nodo de auditoría técnica. Este repositorio documenta fallos de lógica, alucinaciones críticas y comportamientos imprevistos en modelos de lenguaje de código abierto (Open Source).

### 🕵️‍♂️ Áreas de Enfoque
- **Robustez Lógica:** Pruebas de estrés en razonamiento complejo.
- **Seguridad Algorítmica:** Detección de brechas en el alineamiento (jailbreaks/glitches).
- **Optimización de Sustrato:** Configuración de `Modelfiles` para máxima veracidad.

### 🧪 Herramientas del Búnker
- **Motor:** Ollama (Local Deployment)
- **Modelos Auditados:** Gemma 4, Llama 3.x, Mistral.
